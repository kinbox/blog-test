# HTML入门笔记1
### 一.HTML是谁发明的
>“HTML是由WEB之发明者Tim Berners-Lee和同事Daniel W.Connolly于1990年创立之一种标记式语言”

### 二.HTML起手应该写什么
```html
<!DOCTYPE html>
<html lang="zh-CN">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
</body>
</html>
```
